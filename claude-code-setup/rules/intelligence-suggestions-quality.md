# Intelligenz-Vorschlaege: Qualitaetsregeln (KRITISCH)

## Was ein Intelligenz-Vorschlag ist

Ein Intelligenz-Vorschlag ist ein **konkreter Handlungsvorschlag fuer etwas, das NOCH NICHT umgesetzt wurde**.
Er beschreibt eine Verbesserung, die man in Zukunft machen KANN — nicht etwas, das bereits passiert ist.

## Was ein Intelligenz-Vorschlag NICHT ist

- ❌ Eine Zusammenfassung von dem, was gerade gemacht wurde
- ❌ Eine Bestaetigung, dass etwas funktioniert hat
- ❌ Eine Wiederholung von bereits umgesetzten Aenderungen
- ❌ Ein Lob oder eine Bewertung der eigenen Arbeit
- ❌ Eine Erklaerung ohne konkreten Handlungsvorschlag

## Format (PFLICHT)

Jeder Intelligenz-Vorschlag muss **4-5 Zeilen in leichter Sprache** haben:

```
💡 **Intelligenz-Vorschlag N**: [Ueberschrift — was verbessert werden kann]
   Ich habe beobachtet, dass [konkretes Problem oder Ineffizienz].
   Das passiert weil [Ursache].
   Verbesserung: [genauer Vorschlag was gebaut/geaendert/eingerichtet werden soll].
   → Soll ich das umsetzen?
```

## Beispiel: RICHTIG

```
💡 **Intelligenz-Vorschlag 1**: Automatischer Gemini-Key-Check beim Skript-Start
   Ich habe beobachtet, dass der Benutzer nicht merkt wenn ein API-Key ungueltig ist.
   Das Skript zeigt erst beim Diktieren einen Fehler — dann ist es zu spaet.
   Verbesserung: Beim Laden des Skripts den Key einmal testen und bei Fehler
   sofort einen Toast zeigen: "⚠️ Gemini-Key ungueltig — bitte neu setzen."
   → Soll ich das umsetzen?
```

## Beispiel: FALSCH

```
💡 **Intelligenz-Vorschlag**: Diese Session hat 4 neue Regeln erzeugt.
   Die Fehlerrate bei Push-Rejections sollte auf null fallen.
```
↑ Das ist KEIN Vorschlag. Das ist eine Zusammenfassung. Es fehlt der Handlungsvorschlag.

## Pruefung vor jedem Intelligenz-Vorschlag

Bevor ein Vorschlag geschrieben wird, diese 3 Fragen pruefen:
1. **Ist es NEU?** — Wurde das schon umgesetzt? Wenn ja: KEIN Vorschlag.
2. **Ist es eine AKTION?** — Kann der Benutzer "Ja, mach das" sagen? Wenn nein: KEIN Vorschlag.
3. **Ist es KONKRET?** — Weiss der Benutzer genau was passieren wuerde? Wenn nein: genauer formulieren.
