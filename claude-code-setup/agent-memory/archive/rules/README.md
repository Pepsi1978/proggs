# Claude Code Regel-Archiv

Langzeitgedaechtnis fuer alle Regeln die geaendert, ersetzt oder entfernt wurden.

## Warum dieses Archiv existiert
- **Fehlerschutz:** Verhindert dass /self-improve oder andere Agents versehentlich Regeln wieder einfuehren die absichtlich entfernt wurden.
- **Meta-Intelligenz:** Dokumentiert WARUM sich Regeln weiterentwickeln — nicht nur was sich aendert.
- **Stale-Patterns:** Jede archivierte Regel enthaelt Suchbegriffe die in Zukunft NICHT mehr vorkommen sollten.

## Archivierungs-Prozess
1. Wenn eine Regel entfernt oder grundlegend geaendert wird: Neues MD-File hier erstellen.
2. Dateiname: `YYYY-MM-DD-kurzbeschreibung.md`
3. Inhalt: Datum, Grund, alter Stand, Stale-Patterns, betroffene Dateien.
